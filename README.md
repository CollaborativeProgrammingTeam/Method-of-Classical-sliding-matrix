# üìä Comparative Implementation of Classical Sliding Matrix Method in Various Programming Languages

A research project dedicated to the comparative analysis of the implementation of the **classical sliding matrix method** for electricity consumption forecasting in various programming languages.

## üßÆ About the Method and Project

1. **Data Processing Method.** The data processing method, called the "sliding matrix method", allows continuous updating of mathematical model coefficients by adding a new row with actual data at the forecast point and removing the oldest row from the matrix (old information "load").

2. The sliding matrix method is used as a method of statistical analysis of dynamic series in modeling and forecasting electricity consumption.

3. The classical sliding matrix method is implemented in different programming languages for comparative analysis and selection of the best language for creating a forecasting system.

4. Article Title: Comparative Statistical Modeling of Dynamic Series in Forecasting Daily Electricity Consumption in Python, R, C#, C++, Go, Java.
